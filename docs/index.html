Терапевтичний сад для дітей

Терапевтичний сад — це спеціально спроєктований простір на свіжому повітрі, який допомагає дітям відновитися та покращити своє здоров’я за допомогою природи. У саду поєднали зони садівничої терапії, сенсорні маршрути, рухову активність, арт-зону, зону спокою та транспортні інсталяції. :contentReference[oaicite:2]{index=2}

Що таке терапевтичний сад?

Терапевтичний сад — це простір, де діти взаємодіють з рослинами, ґрунтом, водою й звуком для відновлення та розвитку навичок. Тут є сенсорні елементи, грядки для садівництва, місця для спокою і активності. :contentReference[oaicite:5]{index=5}

  • Ігрові та навчальні зони: грядки, мініводойми, сенсорні доріжки.
  • Місця для спокою: альтанки, гамаки, фонтан-каскад.
  • Цілорічна робота: парник/теплиця для розсади.

Переваги

  1. Зниження стресу та тривожності у дітей
  2. Розвиток дрібної моторики та сенсорики
  3. Інклюзія та безпечний простір для всіх

Зони саду

У нашому саду буде 6 зон, кожна виконує певну терапевтичну функцію. Короткий огляд зон: :contentReference[oaicite:6]{index=6}

Зона садівничої терапії

Високі грядки (60–80 см) для дітей в інвалідних кріслах, сенсорні клумби, ягідники, парник для розсади. Частину розсади планується дарувати або продавати на благодійних ярмарках. :contentReference[oaicite:7]{index=7}

Сенсорні маршрути

Доріжки з різними фактурами (пісок, каміння), тактильні панелі і музичні інсталяції для стимуляції органів чуття. :contentReference[oaicite:8]{index=8}

Зона рухової активності

Балансувальні колоди, гойдалки-гнізда, спортивні елементи для фізіотерапії: тренування координації та сили. :contentReference[oaicite:9]{index=9}

Арт-зона

Стіл для малювання, "Стіна мрій", набори для творчості — місце для розвитку уяви і емоційної експресії. :contentReference[oaicite:10]{index=10}

Транспортна зона

Інклюзивно облаштований трамвай і тролейбус для занять, кінопоказів і творчих активностей у будь-яку пору року. :contentReference[oaicite:11]{index=11}

Зона спокою

Альтанки, гамаки та водний елемент (фонтан/джерельце) для затишного відпочинку і зниження стресу. :contentReference[oaicite:12]{index=12}

Кошторис проєкту
